## 🚀 Advanced Techniques

### Technique 1: Boolean Optimization

```
("Director" OR "VP" OR "Head of") 
AND "[sector]" 
AND ("Saudi Arabia" OR "Riyadh" OR "NEOM")
AND NOT ("job" OR "hiring" OR "apply")
```

### Technique 2: Site Stacking

```
(site:linkedin.com/in OR site:linkedin.com/company)
"[role]" "[sector]" "Saudi"
```

### Technique 3: Date Filtering

```
"[role]" "[company]" "Saudi" after:2023-01-01
"appointed" "[position]" "[sector]" "Saudi" after:2024-01-01
```

### Technique 4: File Type Discovery

```
"[company]" "organization chart" filetype:pdf
"[sector]" "supplier list" Saudi Arabia filetype:pdf
"[project]" "tender" Saudi filetype:pdf
```
