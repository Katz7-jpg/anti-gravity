# Project Brief

## Overview

**AI_FACTORY** is a specialized workspace for AI-assisted scientific research and drug development, leveraging the ToolUniverse ecosystem to provide access to 10,000+ scientific tools and APIs.

## Project Goals

1. **Accelerate Drug Discovery** - Provide AI agents with comprehensive tools for target research, drug repurposing, and clinical trial design
2. **Enable Multi-Domain Research** - Support research across oncology, infectious diseases, rare diseases, and antibody engineering
3. **Streamline Scientific Workflows** - Create reproducible, documented research pipelines
4. **Integrate Multiple AI Platforms** - Support various AI coding assistants (KiloCode, Cursor, Claude, etc.)

## Key Stakeholders

- **Researchers** - Scientists using AI tools for drug discovery and research
- **AI Agents** - Automated systems executing research workflows
- **Developers** - Teams extending and maintaining the skill ecosystem

## Success Criteria

- Successful integration of ToolUniverse skills across multiple AI platforms
- Reproducible research workflows with documented outputs
- Comprehensive API coverage for major scientific databases (PubChem, ChEMBL, ClinicalTrials.gov, etc.)

## Constraints

- API rate limits and authentication requirements for external services
- Memory and processing constraints for large dataset analysis
- Compliance with data privacy regulations for clinical data

## Timeline Status

The project is in active development with the Operation-Peanut subdirectory containing the primary skill implementations and configurations.
