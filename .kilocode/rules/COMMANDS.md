# SIMPLE RULES

**Only what you need. No complexity.**

---

## Agent Selection

| What you need | Who handles it |
|---------------|----------------|
| Research, find, discover | Wraith |
| Verify, check, audit | Auditor |
| Execute, close, finalize | Closer |

---

## Task Format

For best results:

```markdown
# Task: [What you need]

## Context
[Background]

## Goal
[Success criteria]

## Deadline
[When needed]
```

---

## Parallel Execution

**Need multiple agents?**

In Kilo Agent Manager:
1. New Session
2. Worktree Mode
3. Select 1-4 agents
4. Run in parallel

---

## Query Format

| Type | Format |
|------|--------|
| Simple | "[question]" |
| Complex | "Wraith: [research task]" |
| Verify | "Auditor: Verify [claim]" |
| Execute | "Closer: Finalize [task]" |

---

## Default Behavior

If you don't specify an agent:
- **Wraith** handles research tasks
- System auto-selects based on context

---
