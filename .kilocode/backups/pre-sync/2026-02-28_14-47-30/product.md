# Product Context

## Product Description

AI_FACTORY is a comprehensive AI-assisted research platform that integrates the ToolUniverse ecosystem to provide scientific computing capabilities for drug discovery, clinical research, and bioinformatics.

## Core Features

### 1. Drug Research & Discovery
- **Target Research** - Identify and validate drug targets with comprehensive data from multiple databases
- **Drug Repurposing** - Find new indications for existing drugs using computational approaches
- **Drug-Drug Interaction Analysis** - Predict and analyze potential drug interactions

### 2. Clinical Development
- **Clinical Trial Design** - Design and optimize clinical trial protocols
- **Pharmacovigilance** - Monitor and analyze drug safety signals
- **Precision Oncology** - Match patients to targeted therapies based on molecular profiles

### 3. Bioinformatics & Computational Biology
- **CRISPR Screen Analysis** - Analyze genetic screens for target identification
- **Structural Variant Analysis** - Detect and interpret genomic rearrangements
- **Rare Disease Diagnosis** - Support diagnostic workflows for rare genetic conditions

### 4. Protein Engineering
- **Antibody Engineering** - Design and optimize therapeutic antibodies
- **Binder Discovery** - Identify protein binders for therapeutic applications
- **Protein Therapeutic Design** - Engineer protein-based therapeutics

### 5. Literature & Knowledge Management
- **Deep Literature Research** - Comprehensive literature search and synthesis
- **Disease Research** - Aggregate knowledge about specific diseases
- **Expression Data Retrieval** - Access gene expression datasets

## User Experience Goals

1. **Seamless AI Integration** - AI agents should be able to discover and use tools without manual configuration
2. **Reproducible Research** - All workflows should produce documented, reproducible results
3. **Multi-Platform Support** - Skills work across different AI coding assistants

## Technical Capabilities

## Integration Points

- **MCP Servers** - Model Context Protocol for tool integration
- **API Keys** - Configured via `.env` file
- **Skills Directory** - Symlinked from user configuration
