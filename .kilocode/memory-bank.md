## 🚀 Quick Start for AI Agents

### New Session Checklist
1. Read [brief.md](memory-bank/brief.md) for project context
2. Check [progress.md](memory-bank/progress.md) for current status
3. Review [tooluniverse-skills-index.md](memory-bank/tooluniverse-skills-index.md) for available tools

### Before Starting Research
1. Review [tooluniverse-overview.md](memory-bank/tooluniverse-overview.md) for ToolUniverse philosophy
2. Check [tooluniverse-workflows.md](memory-bank/tooluniverse-workflows.md) for workflow patterns
3. Use appropriate skill based on task type

### Before Making Technical Changes
1. Review [system-patterns.md](memory-bank/system-patterns.md) for architecture
2. Check [tech-context.md](memory-bank/tech-context.md) for dependencies
3. Follow patterns established in existing code

3. If significant changes, update progress.md decision log
